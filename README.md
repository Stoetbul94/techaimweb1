# TechAim Web

Africa's precision electronic scoring platform, built from the TechAim production blueprint.

## Tech Stack

- Next.js 15 App Router
- TypeScript
- TailwindCSS
- Framer Motion
- Lucide React
- GSAP, Three.js, React Three Fiber dependencies ready for future visual work

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Environment Variables

Create `.env.local` from `.env.example`.

```bash
NEXT_PUBLIC_WHATSAPP_NUMBER=27845173783
NEXT_PUBLIC_CONTACT_EMAIL=arnoldbailie6@gmail.com
```
