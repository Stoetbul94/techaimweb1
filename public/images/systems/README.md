# Systems images

Architecture and infrastructure diagrams. All files use the prefix **`system-`**.

## Pattern

```
system-{descriptor}.png
```

## Examples

```
system-network-architecture.png
system-tablet-architecture.png
system-plc-architecture.png
```

Do not prefix with a product slug (e.g. use `system-plc-architecture.png`, not `ta10x-plc-architecture.png`).
