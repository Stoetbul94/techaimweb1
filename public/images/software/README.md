# Software images

All files in this folder use the prefix **`software-`**.

## Pattern

```
software-{descriptor}.png
```

## Examples

```
software-dashboard-raw.png
software-dashboard-tablet.png
software-dashboard-tablet-black-target.png
software-dashboard-tablet-blue-target.png
software-dashboard-laptop.png
```

Use extra hyphen segments when variants must be distinguished (e.g. target colour, device type).
